## Overview

Home Visit adalah fitur yang memungkinkan customer booking layanan di lokasi mereka (rumah/kantor), bukan di tempat bisnis.

### 2 Mode Home Visit

| Mode | Deskripsi | Kapan Pakai |
|------|-----------|-------------|
| **Simplified (Quota-based)** | Quota global per hari, tanpa assign staff | Bisnis kecil, 1-2 staff |
| **Staff Assignment** | Auto/manual assign staff ke booking | Bisnis dengan banyak staff |

## Setup Awal (Admin)

### Langkah 1: Aktifkan Home Visit di Tenant

```
Settings â†’ Calendar â†’ Home Visit Settings
```

| Setting | Fungsi |
|---------|--------|
| Enable Home Visit | On/Off fitur |
| Daily Quota | Maks booking home visit per hari |
| Time Slots | Jam-jam yang tersedia (misal: 09:00, 13:00, 16:00) |
| Require Address | Wajib isi alamat |
| Calculate Travel Surcharge | Auto hitung biaya transport |

### Langkah 2: Set Service Type

```
Services â†’ Edit Service â†’ Service Location Type
```

| Pilihan | Artinya |
|---------|---------|
| `On Premise Only` | Hanya di salon/kantor |
| `Home Visit Only` | Hanya home visit |
| `Both` | Bisa pilih keduanya |

> **PENTING:** Setelah simpan, field `home_visit_available` otomatis ter-sync untuk tampil di landing page.

### Langkah 3: (Opsional) Setup Staff untuk Home Visit

```
Staff â†’ Pilih Staff â†’ Tab "Home Visit"
```

| Setting | Fungsi |
|---------|--------|
| Layanan Home Visit | On/Off staff bisa home visit |
| Kuota Harian | Maks home visit per hari untuk staff ini |
| Jarak Maksimal | Radius layanan (km) |

### Langkah 4: (Opsional) Setup Jam Kerja Staff

```
Staff â†’ Pilih Staff â†’ Tab "Jam Kerja"
```

Set jam kerja per hari (Senin-Minggu) untuk menentukan ketersediaan staff.

## Quick Reference

### Checklist Setup Home Visit

- [ ] Enable Home Visit di Settings > Calendar
- [ ] Set Daily Quota
- [ ] Set Time Slots yang tersedia
- [ ] Edit Service â†’ Set Service Type ke "Both" atau "Home Visit Only"
- [ ] (Opsional) Setup staff: enable home visit, set kuota, set jam kerja

### Troubleshooting

| Problem | Solusi |
|---------|--------|
| Home visit tidak muncul di landing page | Pastikan Service Type = "Both" atau "Home Visit Only" |
| Tidak ada slot tersedia | Cek quota harian, cek apakah ada staff tersedia |
| Staff tidak bisa di-assign | Cek staff: canDoHomeVisit on, tidak cuti, dalam jam kerja |
| Jam kerja staff tidak tersimpan | Jalankan SQL migration untuk tambah kolom di database |
